# stopwatch-js
Simple stopwatch app built with HTML, CSS, and JavaScript.

## 🚀 Features

- Start, stop and reset the timer
- Displays time in HH:MM:SS format
- Clean and responsive UI

## 📁 Project Structure

stopwatch-js/
├── index.html
├── style.css
└── script.js

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
